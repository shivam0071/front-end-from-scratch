Images

  • You can use Pixabay to find free images, or you can just use Gemini to generate images

  • img is also a self closing tag

  • You can use the 'alt' attribute to describe the image

  • alt is important for SEO and accessibility for people with disabilities

  • We need to specify the width and height of the image in HTML itself because of CLS (Cumulative Layout Shift), If we specify the width and height of the image in HTML, the browser will know how much space to reserve for the image, and it will not cause the page to shift when the image loads.

CSS Notes

General Points

  • Remember this order of specificity (from highest to lowest): Inline styles >> IDs >> Classes, attributes, and pseudo-classes >> Elements and pseudo-elements

  • Gradient is considered as a background image

  • when we use position relative on 2 elements both peer of each other, the first box reserves its space and the second box will be positioned relative to the first box, see file 19_positions.html

  • The absolute doesn't reserve space, It is place accoring to its parent

  • Pseudo Classes are denoted by a colon :, and Pseudo Elements are denoted by a double colon ::

  • The difference between Pseudo Classes and Pseudo Elements is that Pseudo Classes are used to style an element based on its state, while Pseudo Elements are used to style a specific part of an element

CSS Selectors

  • By Default, A body doesn't have a height, we need to set it in the CSS

  • * is a universal selector, it selects all elements

  • Specificity is the order of the selectors, the more specific the selector, the more weight it has

  • Specificity is higher in the following order: ID > Class > Element

  • If we have two selectors with the same specificity, the one that comes last in the CSS file will be applied

  • If we want to select an element that is a direct child of another element, we can use the '>' symbol

  • If we want to select an element that has a specific attribute, we can use the '[]' symbol


Inline, Block and Inline-Block

  • Inline elements take up as much width as their content, and they don't have a height or width and they don't have a margin

  • Block elements take up the full width of the parent element, and they have a height and width

  • Inline-Block elements take up as much width as their content, and they have a height and width

Font - Settings

  • CSS Units - px, pt, em, rem, %, vh, vw, vmin, vmax

  • px - pixels, 1px = 1/96 inch, 0.26mm

  • pt - points, 1pt = 1/72 inch, 0.35mm

  • 1em = 16px, em is relative to the parent element

  • 1rem = 16px, rem is relative to the root element, rem can be used to make the font size of the entire page responsive

  • 1vh = 1% of the viewport height

  • 1vw = 1% of the viewport width

  • 1vmin = 1% of the viewport width or height, whichever is smaller

  • 1vmax = 1% of the viewport width or height, whichever is larger
